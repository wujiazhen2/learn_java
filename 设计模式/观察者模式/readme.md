# 观察者模式

观察者模式(Observer Pattern)：定义对象间的一种一对多依赖关系，使得每当一个对象状态发生改变时，其相关依赖对象皆得到通知并被自动更新

组成：
Subject: 被观察者
ConcreteSubject: 具体被观察者
Observer: 观察者
ConcreteObserver: 具体观察者
